QAPP data generator is an upstream application to push data on to QAPP data files. These data files will be used by QAPP mobile app to show question/answer models. It requires an authentication key to enter data into data files.(Contact us to get authentication key).
Currently this data generator supports following technologies.
  • Java
  • Spring
  • ExtJS
  • SAN(Storage Area Network)
  • AngularJS 1.x

Note: You can use rich text editor in full mode to create/edit your answer.
